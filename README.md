## 📌 Project Overview
This project is created as part of the **CSE0001 – Digital Literacy** course.  
The aim of this project is to develop essential digital skills such as online communication, professional presence, cyber safety, and effective use of digital tools.

The project is divided into five tasks, each focusing on a different aspect of digital literacy.
